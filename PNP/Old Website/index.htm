Testing and analysis of XP/2262 PMTs: Specifically, Gain, Dark Current, and High Voltage curves will be analyzed.  Currently, the setup and programming is being altered to accommodate three of the aforementioned photomultiplier tubes.   The One PMT data analysis is shown below. 

The red peak is known as the pedestal of the histogram, and is an important aspect of analysis, because it gives the true value for zero in the histogram.  After this peak is found, the program fits a Gaussian to the area, shown in the third pane marked “pedestal data”.   The middle pane shows the distribution of data.  The X-axis represents charge and is in units of 25 pC/count.  In order to calculate gain, some knowledge about the initial amount of photoelectrons must be known.  The only practical way to do this is simply to make sure that only one photoelectron hits the 1st dynode of the PMT.  This takes the poissonian distribution outlined in blue.  Once this is known, calculation of gain is simple.

G=Nout/Nin and Nin=1, Nout = Qave/(1.6x10-19C/electron)

The gain for this run was 3.5x108 at 1800V, which is what one would expect.
