Development of stepper motor controls using labview

  • Thorlab- T25xyz servo motor

    

http://www.thorlabs.com/thorProduct.cfm?partNumber=T25XYZ

 

  • The first step was to take the controllers, the power supply and a USB hub, and put them together so that there weren’t wires going every which way. So I did the simplest thing I could imagine and attached it all to a piece of plywood and twist-tie the wires together in a neat manner.

 

  • We first tested the motors using the APT User program. Initially we had to configure the motors using the APT configuration utility, but this proved easy, as we only had to tell it which controllers were controlling what kind of motor.

 

 

 

(The only server setting remains [dwell 5000] as recommended. Not set up any simulators.)

 

With a few test we ascertained that all motors move and show their position. The T25xyz does have an encoder but no limit switches.  Home therefore simply resets the position at the current location to 0000.

 

Labview control

 

The devices come with Labview drivers.  The drivers communicate with the motors using activeX. So labview follows the activeX method to send commands and retrieve status. To make the system work we started Labview

  • Blank VI
  • Modern
    • Container
      • Active X container (drop this on the Front Panel)
      • Right click
      • Insert active x object
        • MG motor control

 

The standard panel for the motor appears on the front panel and a VI appears on the block diagram.  The serial number has to be added to the block diagram VI using the (right click-Create property for ….).

Once you have the VI set you can imbed the VI in a loop

 

The front panel is shown to the right and the block diagram is provided below.  Once the Labview program is started the activeX buttons can be used to control the motor. In addition, more functions can be added to the sequence as MG17Motor blocks only with different functions and different inputs and outputs. The functions can be selected using the right click and select method.
