January 15, 2008-

Today I’m first looking at epics with labview, and I wanted to start a log about it. There are many template files, located C:\VIs\SNS templates\Template.llb.

To enter in a new ip address so epics can talk to that computer, go to control panel-> system-> advanced->environmental variables. To find out what is the ip address of a computer, go to programs->accessories->command prompt, then type in ipconfig.

And yes, I gave each of the pc’s each others Ip addresses, and they talked to each other. The old pc gave me some sort of error when I opened up the level one client template, but they still talked to each other.
